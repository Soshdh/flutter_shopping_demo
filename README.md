# 🛒 Flutter Shopping Demo

![Flutter](https://img.shields.io/badge/Flutter-3.19-blue)
![License](https://img.shields.io/badge/License-MIT-green)

The **Flutter Shopping Demo** project is a portfolio showcase designed to demonstrate how an e‑commerce application can be built using Flutter and Firebase. Inspired by platforms like Amazon and Flipkart, this app focuses on clean architecture, scalable features, and engaging shopping experiences.

---

## ✨ Features (Planned & In Progress)
- 🔑 Authentication → User login/signup with Firebase Auth
- 🛍️ Product Listing → Dynamic product catalog with images and details
- 🛒 Cart Management → Add/remove items, quantity updates
- 💳 Checkout Flow → Order summary and payment integration (future)
- 📦 Order History → Track past purchases
- 🎨 Clean Architecture → UI → Bloc → UseCase → Repo → Source

---

## 🎯 Purpose
This demo project is intended to show recruiters and collaborators that a developer can design and implement production‑ready e‑commerce features. It demonstrates knowledge of scalable architecture, Firebase integration, and modern UI/UX design principles. By building features like product listing, cart, and checkout, the project proves the ability to handle real‑world complexity and transactional workflows.
